## 📌 Executive Summary

2025년은 AI가 "써볼 만한 기술"에서 **"안 쓰면 뒤처지는 직무 필수 도구"** 로 전환된 원년입니다. 기업 AI 도입률이 전년 대비 2배 증가했으며, AI 도구를 적극 활용하는 직장인 비율이 12%(2024년 중반) → 26%(2025년 말)로 급증했습니다.

| 핵심 수치 | 값 |
|-----------|-----|
| AI 활용 직장인 생산성 향상 | +23~33% |
| 일일 절약 시간 | 40~60분/일 |
| 한국 중소기업 AI 도입률 | 31% |
| AI 미도입 기업의 주요 장벽 | 직원 역량 부족 (53%) |
| 글로벌 AI 활용 직장인 비율 | 26% (2025년 말 기준) |

**핵심 인사이트**: 한국은 AI 도입 속도가 상대적으로 느리지만(64% 신중론), ChatGPT 최급성장 시장 중 하나로 OpenAI 서울 오피스가 개설됨. **비개발자 대상 AI 교육 수요가 폭발적으로 증가 중**.

### 1. Claude (Anthropic)

| 구분 | 내용 |
|------|------|
| **주요 모델** | Claude Sonnet 4 / Opus 4 (2025년 5월 출시) |
| **최대 컨텍스트** | 200,000 토큰 |
| **핵심 특장점** | Extended Thinking(사고 과정 공개), 멀티파일 코드 에이전트, 장시간 자율 워크플로우 |
| **요금** | Claude.ai Pro $20/월 |

**주요 기능**:
- **Extended Thinking Mode**: AI가 답변 전 추론 과정을 단계별로 보여주는 투명한 사고 방식
- **Claude Opus 4**: 수천 단계의 복잡한 작업을 거의 하루 종일 자율 수행 가능 (업무 에이전트)
- **Claude Sonnet 4**: 64K 출력 토큰 지원, 코드 작업 신뢰도 향상
- **Tool Use (MCP)**: 외부 서비스와 연동해 실제 작업 수행 (파일 읽기/쓰기, DB 조회, API 호출)
- **TAU-bench 기준** 소매 업무 자동화 81.2%, 항공 업무 58.4% — 업무 에이전트로서 최상위권

**비개발자 실무 활용 케이스**:
```
✅ 긴 계약서/보고서 요약 (200K 토큰 = 책 한 권 분량 처리)
✅ 마케팅 카피 A/B 변형 생성
✅ 회의록 → 액션 아이템 자동 정리
✅ 이메일 초안 작성 (톤앤매너 지정)
✅ 데이터 분석 리포트 해석 및 시사점 도출
```

### 5. Cursor

| 구분 | 내용 |
|------|------|
| **형태** | VS Code 기반 AI 코딩 에디터 |
| **요금** | $20~40/월 |
| **핵심 특장점** | Composer 모드, 멀티파일 에이전트, Max Mode(1M 토큰) |

**주요 기능**:
- **Composer Mode**: 자연어 명령으로 여러 파일에 걸친 코드 자동 수정
- **Claude 4 / GPT-4o 등 프론티어 모델 접근**: 크레딧 방식으로 최신 모델 사용
- **Background Agents**: 백그라운드에서 테스트·빌드·리팩토링 자동 수행
- **Max Mode**: 최대 100만 토큰으로 대규모 코드베이스 처리

**비개발자 관련성**: 노코드 앱 개발에 진입하는 비개발자들이 Cursor를 이용해 간단한 웹앱, 자동화 스크립트를 자연어로 생성하는 트렌드 증가
